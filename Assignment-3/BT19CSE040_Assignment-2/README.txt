For executing any of the source codes follow the following steps:

1. Install flutter from Flutter Assignment.pdf
2. Download the source code from the PoC pdf.
3. Open it using the VSCode editor and connect to a device.
4. In VSC - 
	-> Menu File → Preferences → Settings
	-> Search for "Flutter run additional args"
	-> Then click Add Item
	-> Now type --no-sound-null-safety
	-> Click OK
Then,
	Menu File → Run → Run without debugging or start debugging.
